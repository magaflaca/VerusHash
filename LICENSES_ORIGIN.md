# License and origin

This package contains a standalone extraction of VerusHash code from the provided VerusCoin source archive.

The extracted VerusCoin source is distributed under the MIT license included in `LICENSE`.

The Haraka implementation included in `crypto/haraka.*` and `crypto/haraka_portable.*` carries the MIT license from its original implementation.

The CLHash-derived VerusHash implementation in `crypto/verus_clhash.*` carries the Apache-2.0 notice from the VerusCoin source comments and the original CLHash work.

The `sse2neon.h` compatibility header is included from the provided VerusCoin source archive to support ARM and ARM64 builds.
